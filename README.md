# AWS EC2 Ubuntu Server Project

## Project Overview
This project demonstrates how to provision and configure an Ubuntu EC2 server on AWS.

## Tasks Completed
- Launched an Ubuntu EC2 instance
- Configured SSH access using key pairs
- Configured security groups
- Worked with public subnet settings
- Connected remotely using SSH

## Tools Used
- AWS EC2
- Ubuntu Linux
- SSH
- Git Bash

## Commands Used

```bash
chmod 400 feranz.pem
ssh -i feranz.pem ubuntu@52.205.81.20
sudo apt update
```
